# ecommerce-platform

A microservices-based e-commerce platform built to learn and demonstrate:

- System Design (HLD & LLD)
- Event-driven architecture
- API Gateway & Authentication
- Kafka, Redis, PostgreSQL
- Docker & SAP BTP

## Services
- gateway-service
- auth-service
- user-service
- product-service
- cart-service
- order-service
- payment-service
- notification-service
